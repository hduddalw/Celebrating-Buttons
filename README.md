# Introduction 
HTML Starter Template

# Features
1. HTML Partials
2. Bootstrap Grid
3. Actions folder for php action files
4. SCSS Compiler
5. JS Minifier
6. Node Modules JS to import
7. Node Modules CSS Vendor file
8. GULP LOCALHOST

# Steps for Setup

### Install NPM Modules and Dependencies
```
yarn install
```

### Compile the Gulp File
```
gulp
```

### Watch Files to update gulp
```
gulp watch
```

### Run Localhost
```
gulp localhost
```
